# ARTPS: Depth-Enhanced Hybrid Anomaly Detection and Learnable Curiosity Score for Autonomous Rover Target Prioritization

- **Author**: Poyraz BAYDEMİR
- **Category**: Computer Science > Computer Vision and Pattern Recognition (cs.CV), Computer Science > Artificial Intelligence (cs.AI)
- **Keywords**: Autonomous Exploration, Mars Rover, Anomaly Detection, Depth Estimation, Curiosity Score, Computer Vision

## Abstract
This work presents an integrated approach for on-site prioritization of scientifically interesting targets in autonomous exploration scenarios on planetary surfaces. The proposed ARTPS architecture consists of single-image depth estimation, multi-component anomaly fusion of image and depth cues, engineering-focused target localization, and learnable curiosity score components.
